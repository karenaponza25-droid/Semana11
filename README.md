# Semana11
Semana 11

Base de datos, autenticación, interfaz básica y trazabilidad

Objetivo

Comprender cómo integrar una base de datos, la autenticación de usuarios, una interfaz básica y la automatización dentro de una aplicación Node.js.

Archivos de la actividad

* preguntas-semana11.txt
* modelo-datos.txt
* autenticacion.txt
* interfaz.txt
* reflexion-semana11.txt

Actividades realizadas

* Respondí las preguntas de selección múltiple.
* Diseñé un modelo de datos con dos entidades.
* Expliqué la importancia de la autenticación y la protección de datos.
* Describí una interfaz básica para la plataforma.
* Escribí una reflexión sobre la importancia de la base de datos, la autenticación y la seguridad de la información.
